
## Project-title : Keen-keeper

    Description: Keen Keeper is a modern relationship management web application built with Next.js that helps users track, manage, and maintain meaningful connections. It provides a clean dashboard, interaction timeline, and smart status insights to ensure you never lose touch with important people. 

    Technologies Used: •Next.js
                       •React.js
                       •Tailwind Css
                       •DaisyUi
